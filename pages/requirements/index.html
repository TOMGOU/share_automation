Requirements

python 项目自动生成requirements.txt文件

方法一(全部类库)：
pip freeze > requirements.txt
方法二：
pip install pipreqs
pipreqs ./
pipreqs ./ --encoding=utf8

安装requirements.txt中的类库

pip install -r requirements.txt

pip install --upgrade -r requirements.txt
